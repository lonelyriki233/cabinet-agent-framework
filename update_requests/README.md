# CAF 更新需求目录

本目录用于记录 CAF 框架本体的更新、维护、蓝图、任务拆解和变更报备。

以后凡是涉及 CAF 框架本体升级，都优先在这里留下记录：

- 新需求
- 蓝图版本
- 当前缺口
- 任务拆解
- 已完成更新
- 待验证事项
- 重要决策
- 返工原因

约定：

1. 不把具体业务项目写成 CAF 本体目标。
2. CAF 的主线是 Agent Project Harness：支持复杂项目开发、维护、审查、迭代。
3. 每次重要更新前，先确认是否需要新增或修改本目录下的蓝图/任务文件。
4. 每次重要更新后，把变更内容、验证结果、遗留问题写回本目录。

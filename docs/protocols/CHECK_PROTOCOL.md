# CHECK_PROTOCOL.md - 谁检查 worker，如何检查

## 三层检查

1. Worker 自检：是否完成任务包、是否产出文件、是否跑了本领域验证。
2. 中书检查：任务是否满足领袖目标，是否有跨 worker 依赖，是否记录状态。
3. 领袖检查：是否满足用户需求，是否需要继续、停止、人工确认。

## 必检内容

- 是否有任务包。
- 是否有本地成果。
- 是否有 worker 更新日志。
- 是否触碰 forbidden scope。
- 是否满足验收标准。
- 是否需要 skill 更新。
- 是否有未处理风险。

## 检查目的

不是证明 agent 永远正确，而是让错误可发现、可定位、可回滚、可复盘。
